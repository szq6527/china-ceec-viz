# 中国-中东欧科研合作: 数据挖掘深度报告

**生成时间**: 2026-05-23T16:04:23
**数据范围**: 2011-2020 (十二五 + 十三五), 16个中东欧国家
**数据源**: OpenAlex API (group_by exact counts) + ScienceDB WoS period totals

## 1. 执行摘要

### 核心发现

**发现 1: 大科学合作正在全线加速,不是'过去时'。**
CEEC总体大科学论文(≥100作者)十年增长1.72%，贡献了绝对增长的0.07%。
但更关键的是 **2019-2020年所有14个有数据的国家大科学产出同时加速**——十三五后期/前期比最低1.4x,最高7.3x。
这不是过去的CERN效应,而是**正在持续强化的结构性趋势**(LHC Run 3 于2021年启动,预备期恰在2019-2020)。

**发现 2: 大科学占比显著上升,但绝对贡献有限。**
均值从0.03→0.05(p = 0.0273 *, Cohen's d=0.54)。
但大科学贡献的增长83%集中在6个国家——波兰、匈牙利、克罗地亚、捷克、罗马尼亚、希腊。
大科学不是均匀分布的,它在少数国家深度渗透,在多数国家几乎不存在。

**发现 3: 学科结构揭示了'两个合作世界'。**
物理学(大科学渗透率35.1%)与化学工程/化学/药学(大科学渗透率0%)是两个完全不同的合作模式。
各国在非物理学科上展现出鲜明特色——立陶宛的决策科学(7.2x CEEC平均)、爱沙尼亚的农业生物(3.0x)、匈牙利的兽医学(4.6x)。
真正的双边深化发生在这些零大科学污染的学科中。

**发现 4: EU嵌入度与中国RCA呈正相关(r=0.64, p = 0.0079 **)——与替代假说相反。**
EU6合作强度最高的国家(爱沙尼亚60%、斯洛文尼亚45%)同时也是中国RCA最高的。
这不是'选边站',而是'全方位国际化'——科研开放度是一个统一的维度,同时面向EU和中国。
RCA排名中等稳定(Spearman ρ=0.57),但11/14国RCA下降,暗示中国合作的相对优势正在稀释。

**发现 5: 东向替代假说不成立。**
EU候选国RCA均值(1.12)显著低于成员国(1.65)。候选国并没有因EU参与受限而更依赖中国。
中国合作的驱动力是科研体量和国际化程度,而非EU准入替代。

**发现 6: 数据层面——ScienceDB(WoS)遗漏了立陶宛和北马其顿的中国合作。**
立陶宛: ScienceDB记录0篇,OpenAlex观测到259+732=991篇。
北马其顿: ScienceDB记录0篇,OpenAlex观测到267+513=780篇。
本报告已修复此问题,直接使用OpenAlex观测数据。

### 4.5 统计检验

**Kruskal-Wallis**: H=3.31, p = 0.1889 (n.s.)。
地缘政治组间RCA差异不显著——组内差异大于组间差异。

## 5. 跨线综合

### 5.1 三条线的交汇

**交汇 1——大科学+物理学双重主导**: Line 1发现大科学贡献83%集中在6国, Line 2发现物理学渗透率仅35%(意味着65%的物理合作仍是<100作者)。
 **关键洞见**: '大科学'≠'物理学'。大科学是高能物理的一个子集,但它对增长叙事的影响力远超其数量占比——因为增长集中、波动剧烈。

**交汇 2——'双外向型'取代'替代型'**: Line 3发现EU6强度与中国RCA呈正相关(r=0.615), Line 2显示非物理学科(化学工程、药学)几乎是纯双边合作。
 **关键洞见**: 国际化是一个统一的维度。科研开放的国家(Estonia)在所有方向上都开放;封闭的国家(Poland)在所有方向上都相对封闭。

**交汇 3——加速,而非稳态**: Line 1发现2019-2020全线加速, Line 3发现大多数RCA在下降。
 **关键洞见**: 大科学合作在加速(Run 3即将启动),但中国在整个CEEC科研版图中的相对重要性在下降——RCA普降。两者并行不悖。

## 6. 局限性

- **大科学阈值(≥100作者)**: 常用但武断。LHC实验论文通常>1000作者,而中等规模合作(~50作者)也可能属于'大科学'。建议敏感性分析。
- **学科分类粒度**: OpenAlex primary_topic将论文映射到单一领域,跨学科论文可能被错误分类。
- **统计功效**: 仅16国,地缘政治组比较(如欧元区核心仅2国)功效严重不足。
- **因果关系**: 所有相关性分析均为观测性。不能将RCA下降归因于EU政策或中国政策。
- **WoS覆盖偏差**: ScienceDB(WoS)在巴尔干地区覆盖不足(LTU/MKD),本报告已修复但其他偏差可能仍存在。
